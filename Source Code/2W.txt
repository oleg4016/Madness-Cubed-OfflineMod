Всё в двух файлах:

1. Характеристики персонажа (Здоровье/Броня/Скорость/Прыжок/Защита) ShopCore.cs:242–247 — массив charLvls:

5 строк (здоровье, броня, скорость, прыжок, защита)
В каждой строке группы по 5 чисел: [X, цена_баксы, цена_золото, треб_уровень, значение_стата]

Пример строки здоровья: 0, 500, 0, 0, 100, 2, 1000, 0, 0, 105, ... → Ур.0: цена 500 баксов, треб. ур.0, значение 100 HP

2. Покупка/улучшение оружия WeaponPriceData.cs:

WeaponBuyPrices[74,2] (строка 9): { цена_баксы, цена_золото } для каждого оружия
WeaponUpgradePrices[74][] (строка 87): [ базовая_цена, q0, q1, q2, q3 ], где отрицательные = золото, положительные = баксы

Всё в ShopCore.cs, метод ApplyDefaultPrices():

Цены одежды — строка 204, массив clothesPrices:

{ цена_баксы, цена_золото }
91 элемент (ID 0–90). Номер строки = ID одежды.
